## MusicGPT

[gabotechs's MusciGPT Repo](https://github.com/gabotechs/MusicGPT)

You can generate music using natural language prompts locally on MusicGPT.

## Install

[Official Instruction](https://github.com/gabotechs/MusicGPT)

## Usage

Execute the following command in terminal:

```shell
musicgpt
```
