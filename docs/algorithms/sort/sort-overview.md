## Sort Algorithms

| Algorithms | Time Complexity |
| --- | --- |
| Bubble Sort | $O(n^2)$ |
| Insertion Sort | $O(n^2)$ |
| Selection Sort | $O(n^2)$ |
| Quick Sort | $O(n \log n) \rightarrow O(n^2)$ |
| Merge Sort | $O(n \log n)$ |
| Heap Sort | $O(n \log n)$ |
| Bucket Sort | $O(n + n^2/k + k) \rightarrow O(n^2)$
| Radix Sort | $O(n)$ |
| `std::sort()` | $O(n \log n)$
