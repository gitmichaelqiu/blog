# Ⅰ. 地图信息的获取、表达和理解
## 从地图和图表中获取信息
```text
《人文地理 上册》 P2~4, P14~20, P45~52
```

### 地图三要素
> 方向，比例尺，图例 & 注记 [^1]

[^1]: 合为一个要素

#### 方向🧭
???- tip "正确表达某地的经纬度"
    Point A (n° N/S, n° W/E)

    如：A点（30° N, 45° W）

!!! warning
    站在地球的南(北)极点，前后左右都是北(南)方

#### 图例 & 注记🏷️
**图例和注记的区别**

> **图例：**对地图上**各种符号**和**注记**的说明，一般在地图<u>外面</u>

> **注记：**指地图**上**的**文字说明**，一般地图<u>里面</u>

### 从地图🗺️上获取信息
#### 等高线地形图
???+ tip 等高距和比例尺的关系
    ![](geography-att/等高线地形图.jpg)

    如上图，图幅和图中**等高距相同时，比例尺越小，坡度越缓**，反之同理

    $\therefore$ 坡度最缓的是C

???+ "山脊和山谷的判定"
    **山脊：**等高线向**低处**凸出
    **山谷：**等高线向**高出**凸出 <u>(可以发育河流)</u>

### 从地球仪🌎上看世界
#### 经线和纬线
> 经线又名子午线🌐

???- tip "经线纬线的长度和形状"
    每条经线呈半圆状，每条长度相等

    每条纬线呈整圆状[^2]，赤道最长

[^2]: 极点除外

#### 经度和纬度
| 项目 | 经度 | 纬度 |
|---|---|---|
| 度数起始线 | 本初子午线 | 赤道 |
| 划分方法 | 以本初子午线为界，向东东经，向西西经，各180° | 以赤道为界，向北北纬，向南南纬，各90° |
| 半球划分 | 东半球🌏：20° W - 160° E | 赤道为界 |

!!! danger
    东西半球分界线为<u>20° W, 160° E</u>

???- quote "五温带 & 低中高纬度"
    ![](geography-att/五带&纬度带.jpg)

    据图可知，我国大部分地区位于<u>北温带，中纬度</u>；小部分地区位于热带，低纬度

!!! tip ""
    经度相差15°，时间相差1h

## 描述区域的位置、自然环境和人文环境的特点
```text
《人文地理 上册》P100~104
```

### 描述地区的自然环境🌲特点
!!! abstract ""
    地形、气候、河流、自然资源等

#### 描述方法
???- info "地形"
    - 海拔
    - 地面起伏
    - 地形种类和分布

???- info "气候"
    - 气温
    - 降水
    ???+ tip "归纳气候特征"
        季节+气温特点+降水特点

???- info "河流"
    - 年径流量
    > 年降水量
    - 径流量季节变化
    > 降水量季节变化
    ???- example "亚热带季风气候"
        $\because$ 夏季多雨，冬季少雨

        $\therefore$ 河流径流量夏季大，冬季少
    - 流速
    > 等高线
    - 流向
    > 地势
    - 水能
    > 地势落差，径流量
    - 水位
    > 河流宽度，径流量
    - 含沙量
    > 土质，人为因素
    - 结冰期
    > 气温
    - 水系特征描述
    > 河网(密集/稀疏)、河湖数量、长度、入海口、流域面积等

???- info "资源"
    **常见资源：**土地资源、矿产资源、生物资源、水资源、太阳能、风能等

    **常见表述：**"土地资源广阔，土地肥沃" "煤炭资源丰富" "水资源短缺"等

???- info "耕地"
    分为以下2类：
    
    1. **水田：**水资源丰富、地势低平的地方，一般以水田为主

    2. **旱地：**水资源不足，地势又相对较高的地方，一般以旱地为主

### 描述地区的人文环境🗿特点
!!! abstract ""
    人口、民族、城市和乡村、风俗习惯等

#### 描述方法
- 生活特色
> 衣食住行，传统文化，风俗习惯
- 农业生产
> 农业生产类型：种植业、林业、畜牧业、渔业等
> 种植业：主要种植的农作物种类、耕作制度、生产条件
- 其他生产特色
> 制造业、高科技产业、金融业、服务业等

### 感受区域的发展
!!! abstract ""
    一个区域的发展一般可以通过该地的<u>政治、经济和文化发展水平</u>等几个方面来认识

???+ example "南非🇿🇦"
    南非依靠自然条件的优势，**因地制宜**地发展经济。

    例如，充分利用其**矿产资源**优势，大力发展**采矿业**和**制造业**；

    利用丰富的**野生动物资源**，发展**旅游业**，等等。

### <font color = red>【重点提醒】</font>
???+ warning "影响农业生产的因素"
    ???+ info "光"
        - **纬度：**纬度低，太阳辐射量丰富
        - **天气：**晴天多，光照时间长
        - **海拔：**海拔高，空气稀薄，透明度好，太阳辐射量丰富
    ???+ info "热"
        - **季节：**夏季温度高，热量条件好
        - **纬度：**低纬度地区热量条件好
        - **地形：**海拔高，气温低，热量不足；盆地地形不易散热，热量条件较好，东西走向和东北一西南走向的山脉，大大减弱了冬季寒潮和冷空气的入侵程度等
    ???+ info "水"
        - **时间：**季节变化角度
        - **空间：**如“东多西少”“南多北少”“由东南沿海向西北内陆逐渐减少”等
        - **影响降雨因素：**
            1. 夏季风（一般情况下，受夏季风影响明显的地方，降水较多）
            2. 海陆位置（一般情况下，沿海地区降水多，内陆地区降水少）。
            3.  地形（一般情况下，湿润气流遇到高山阻挡时，迎风坡降水多，背风坡降水少）
    ???+ info "土"
        1. **地形：**如平原，平坦辽阔，适宜种植农作物；丘陵适合种植茶叶、柑橘等经济作物
        2. **土壤：**如河流冲积平原，土壤肥沃，适合农作物的生长；东南丘陵地区的酸性土壤适宜种植茶树等作物；碱性土壤则适合种植喜碱性的作物

???+ warning "影响工业生产的因素"
    地理位置、资源、交通、市场、科技、人才、劳动力、资金、首础设施、历史条件、政策支持等
    ???+ warning "珠三角地区"
        还要考虑是否**靠近港澳**，是否**地处侨乡**。这里有以引外来资金尤其是滤澳资全的有利条件，还有发达的市场经料广阔的市场等。

### 地形多样
???- quote "五种基本地形"
    | 类型 | 海拔 | 地表特征 |
    |---|---|---|
    | **山地** | 500米以上 | 峰峦起伏,坡度陡峻。有的呈长条状延伸,形成山脉。 山脉排列有序,脉络分明,仿佛是"大地的骨架" |
    | **高原** | 500米以上 | 地表起伏不大,但边缘处比较陡峭;也有的高原表面 山峦起伏,凹凸不平 |
    | **丘陵** | 200~500米 | 起伏和缓 |
    | **平原** | 200米以下 | 地面平坦或起伏较小,主要分布在大河两岸和濒临海洋的地区 |
    | **盆地** | 无标准 | 四周被群山环绕,四周高,中间低 |

### 主要气候类型
???+ abstract "主要气候类型"
    - **热带南林气候：**终年高温多雨
    - **热带章原气候：**终年高温 分干湿两季
    - **热带季风气候：**终年高温 分旱雨两季
    - **热带沙淡气候：**终年炎热干旱
    - **亚热带季风和湿润气候：**夏季高温多雨 冬季温和少雨
    - **地中海气候：**夏季炎热干燥 冬季温和多雨
    - **温带海洋性气候：**冬温夏凉 全年降水均匀
    - **温带季风气候：**夏季高温多雨 冬季寒冷干燥
    - **温带大陆性气候：**冬冷夏热 全年降水较少
    - **寒带气候：**终年寒冷干燥

    ???+ tip 高原山地气候的特点
        寒冷，很多地方终年冰雪覆盖，气侯重直变化明显。主要分布在高大的高原、山地等地区，如青藏高原、阿尔卑斯山、安第斯山脉等。

# Ⅲ. 生活在不同区域里
???+ abstract "典型国家汇总"
    ???+ info "美国🇺🇸"
        ???- question "美国商品农业发达的有利条件(原因)"
            1. **自然条件：土地**辽阔，平原**地形**，**地势**低平，**土壤**肥沃；温带大陆性**气候**，气候适宜；密西西比河自北向南贯穿平原，**水源**充足，灌溉便利。
            2. **人文条件：**经济发达，农业科技水平高，农场规模大，地广人稀，机械化生产，效率高，产量大，产品储存条件好，交通四通八达等等。

    ???+ info "瑞士🇨🇭"
        ???- question "瑞士成为旅游热区的有利条件(原因)"
            1. 自然条件：
                1. 地形多样，海拔高，有冰川和雪山，风景优美，景观奇特。
                2. 夏天成观光、避暑、登山胜地；冬天发展滑雪运动的条件十分理想，温泉资源也很丰富。
            2. 人文条件：
                1. 旅游内容丰富多彩，配套设施完备、精良，服务周到、细致入微等。
                2. 拥有发达的铁路和公路；游客乘火车可直接到达山峰观看冰川奇景；许多地方有电缆车直达山巅；还有直升机为遇到危险的游人提供救助。
                3. 有古老的城镇和纯朴的乡村生活等。
